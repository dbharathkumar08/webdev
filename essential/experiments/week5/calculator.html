Calculator using Angular JS

About The Experiment

In this experiment, I have made a small calculator which calculates square and cube of given input using Angular. Here the CalculatorController has two functions for square and cube. The scope variable is number which is multiplied with itself depending on whether the function is square or cube.

Code Snippet

app.controller('CalculatorController', function ($scope) {
    $scope.clear = function () {
        $scope.number = '';
        $scope.answer = '';
    }
    $scope.doSquare = function () {
        $scope.answer = $scope.number * $scope.number;
    }
    $scope.doCube = function () {
        $scope.answer = $scope.number * $scope.number * $scope.number;
    }
});

Description

Here the code for controller is shown. It contains 3 function. clear() function clears out the answer and input by resetting their values to null. doSquare() function multiplies the scope variable number with itself twice and stroes the value in scope variable answer. doCube() function calls the doSquare() function on number and multiplies the answer with number again and stores the value in scope variable answer.
