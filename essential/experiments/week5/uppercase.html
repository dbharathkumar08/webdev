Uppercase Filter

About the experiment

In this experiment, I have used Angualr js's filter to covert the text to upper case. Here whatever user inputs in the given tetxarea is converted to uppercase through uppercase filter.

{{text | uppercase}}

Code Snippet

<textarea ng-model="text" rows="7" cols="50"></textarea>
<p>
       {text | uppercase}
</p>

In this code I have shown the usage of the filter. This is the filter for uppercase. Here the input is bound to scope variable text. That text is converted to uppercase by using the uppercase filter.
