Editable Content Using Angular js

About The Experiment

In this experiment I have used the inline editing property of Angular js. Here the user can click on the item to make it editable. After editing it, user can save changes to make them reflect in DOM or else he can discard them. Here I have used the 'show' and 'hide' property of angular to display the text box on click. For editing, I have made objects editable.

Click on The content to make it editable

Code Snippet

<h1 class="center" ng-click="editable = 'name'" ng-hide="editable == 'name'">{{person.name}}</h1>
<span ng-show="editable == 'name'">
    <form class="form-inline">
        <input type="text" size="30" name="name" ng:required ng-model="name" id="data">
        <button class="btn btn-success"  ng-click="save();editable = ''"">Ok</button>
        <button class="btn btn-warning" ng-click="editable = ''">Cancel</button>
    </form>
</span>

There are 2 controllers I have used here. Profile and Editor.

  • Profile maintains the data about person which is bounded with scope variable.
  • Editor makes it possible to edit the data. Also it has function for save(), which allows user to save changes he made to person name.
