Place Search Autocomplete

About The Experiment

In this experiment I have used the Google maps API for providing the option for autocomplete. I need the user to enter any city he has visited, so I have restricetd the search to cities only.

Code Snippet

$(function () {
    var options = {
        types: ['(cities)']
    };

    var input = document.getElementById('cityname');
    var autocomplete = new google.maps.places.Autocomplete(input, options);
});

Description

Here in this code I have shown the usage of maps API for autocompletion. The text entered in the input box is stored in a variable and that variable is passed to API for Autocomplete along with options constraint. This constraint limits the search to city names only.
