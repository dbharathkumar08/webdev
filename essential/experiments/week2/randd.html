Image Drag And Resizing

About the Experiment

Here I have created the code to resize a given image. User can drag the image on screen anywhere as well he can resize the image by pulling it from corners. I have made use of javascript resizable() and draggable functions for that. I have included the jQuery and javascript files from Google CDN to make the experiment run.

Try Dragging And Resizing The Image!!!

Code Snippet

<script type="text/javascript">

            $(document).ready(function () {
            $(".resizebleImage").resizable().parent().draggable();
            });
            </script>

The resizableImage class will call the function upon drag and resize events which will in turn call the inbuilt javascript functions for that.
