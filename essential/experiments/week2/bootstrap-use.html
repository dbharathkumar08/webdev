Having Fun With Bootstrap

About The Experiment

In this experiment I have tried my hands on Bootstrap framework. Here I have created an educational portal fot HTML,CSS,JavaScript and given it various CSS effects through Bootstrap. I have made use of:

  • nav nav-tabs for Navigation purpose
  • dropdown-toggle for creating dropdown menu
  • embed-responsive-item to include videos in webpage
  • panel classes for creating Collapsable items

HTML5 is the latest evolution of the standard that defines HTML. The term represents two different concepts: It is a new version of the language HTML, with new elements, attributes, and behaviors, and a larger set of technologies that allows more diverse and powerful Web sites and applications. This set is sometimes called HTML5 & friends and often shortened to just HTML5.

CSS3 is the latest evolution of the Cascading Style Sheets language and aims at extending CSS2.1. It brings a lot of long-awaited novelties, like rounded corners, shadows, gradients , transitions or animations, as well as new layouts like multi-columns, flexible box or grid layouts. Experimental parts are vendor-prefixed and should either be avoided in production environments, or used with extreme caution as both their syntax and semantics can change in the future.

Code Snippet

This is the HTML code for creating Navigational tabs and dropdown menu. I have made use of nav navbar-tabs class with navbar-inverse to make it black. To create dropdown menu I have used dropdown toggle class with dropdown menu 

<ul class="nav nav-tabs navbar-inverse" role="tablist">
<li class="dropdown">
<a class="dropdown-toggle" data-toggle="dropdown" href="#">
Tutorials <span class="caret"></span></a>
<ul class="dropdown-menu" role="menu">

This is the HTML code for creating Embedded video inside a webpage. 

<iframe class="embed-responsive-item" src="https://www.youtube.com/embed/rvFdP8taVQo" height="300" width="350"></iframe>

This is the HTML code for creating a collapsable panel. I have used the main panel clas and inside that I have made use if collapse class to create a collapsable item. 

<div class="panel panel-default">
  <div class="panel-heading">
    <h4 class="panel-title">
    <a data-toggle="collapse" data-parent="#accordion" href="#collapse2">CSS3 Course</a>
    </h4>
  </div>
 <div id="collapse2" class="panel-collapse collapse">
<div class="panel-body">
