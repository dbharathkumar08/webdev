Responsive Form Design without Bootsrap

About the Experiment

Here I have created a responsive form design with a fixed header by using simple CSS. The form elements like textbox, columns will adjuxt themselves when the size is reduced from Desktop to Tablet to Mobile device. Also I have created a resonsive menu which somewhat changes to dropdown menu when the size is reduced. Also the Image in the form changes its size according to the screen size.

Code Snippet

@media screen and (max-width: 500px){
            .field-label {
                float:none;
            }
            .container {
                width: 300px;
                margin: auto;
            }
            .row {
                padding-top:100px;
            }
            body header nav {
                display: none;
            }
            #menu-link, #small-menu {
                display:block;
            }
            main{
                padding-top:100px;
            }
            body header{
                padding-top: 0px;
                margin-top: 0px;
                position:fixed;
                width:600px;
                background-image:url("../../images/gbg.jpg");    
            }    
            }

This is the code snippet (CSS) for mobile screen size. When the size is max 500px, the position of header is fixed and the background is same as the original background so that the header is not transparent and user can scroll up and down. Also for this size, the navigation links are hidden and instead a menu link is shown which acts like a dropdown menu. The body row is pushed down the fixed header.
