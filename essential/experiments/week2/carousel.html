Carousel

About The Experiment

In this experiment I have tried creating an image scroller also known as Carousel using Bootstrap. This experiment sutomatically scrolls images at interval of 5 secs. user can also go to next image manually using the side buttons.

Code Snippet 

        <!-- Indicators -->
        <ol class="carousel-indicators">
        <li data-target="#carousel-example-generic" data-slide-to="0" class="active"></li>
        <li data-target="#carousel-example-generic" data-slide-to="1"></li>
        <li data-target="#carousel-example-generic" data-slide-to="2"></li>
        </ol>

This is the code snippet for small indicators at the bottom of each image. The indicators are declared using an ordered list. I have used the bootstrap class carousel-indocator for it. 

        <!-- Wrapper for slides -->
        <div class="carousel-inner">
        <div class="item">
        ...
        </div>
        </div>

The wrapper contains the slider images and the captions. 

        <!-- Controls -->
        <a class="left carousel-control" href="#carousel-example-generic" role="button" data-slide="prev">
        <span class="glyphicon glyphicon-chevron-left"></span>
        </a>

The controls are the left and right arrows at the ends of the slider images. This allows users to manually toggle back and forth between the sliders.
