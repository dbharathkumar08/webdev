Introduction to Angular js

About The Experiment

This is a basic experiment which gives an intro to angular js functionality. It shows how data is updated dynamically without rendering the whole page again. It shows how the DOM changes due to Angular. Here the user has to type his/her name in the input box and that name will appear in the begining of the paragraph.

Please input your name in the field

{{hello}} , Hello from Angular!! Good luck!

Code Snippet

var app = angular.module("HelloApplication", []);
app.controller("control", function ($scope) {
    $scope.hello;
})

Description

Here this code snippet shows the javascript, which shows the controls for module and controller. Here for the controller module, a function with $scope variable is called. This function binds the value of scope to the value of input. 

<div ng-controller ="control">
        <h1>Please input your name in the field</h1>
        <input ng-model ="hello" placeholder="Enter your name"/>
        
        <p>
            <span class="name">
            {{hello}}
                </span>, Hello from Angular!! Good luck!</p>
    </div>

Description

Here "HelloApplication" is define as ng-app which tells the angular.js to treat as angular application. Inside ng-app angular will search for a controller which is indicated by ng-controller. Inside that there is ng-model. The value that model has, is reflected in the $scope variable. So here "hello" is scope variable whose value will be changed according to value iin input box.
