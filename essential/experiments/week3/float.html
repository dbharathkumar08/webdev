Code Snippet

function FloatMenu(){

    var scrollAmount=$(document).scrollTop();

    var newPosition=menuPosition+scrollAmount;

    if($(window).height()<$fl_menu.height()+$fl_menu_menu.height()){

        $fl_menu.css("top",menuPosition);

    } 
    else {

        $fl_menu.stop().animate({top: newPosition}, $float_speed, $float_easing);

    }

}

Here the function calculates the top scrolling amount and with reference to that, relative scroll amount is calculated. If that amount is greater than preset bar, menu is animated to new position.
