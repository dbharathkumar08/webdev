Form Validation Using jQuery

About The Experiment

Here I have created a form on which various validation tricks have been applied. They are as following:

  • When you hover over the question mark near password field, it shows you requirements of ideal password
  • When you start typing in the password, it shows you the strength of the password
  • When you click on register button it matches both passwords and shows you wheather they match or not


?
  • Password should be:
  • more than 6 chars long
  • contain alphanumericals
  • contain atleast one special char



Code Snippet

        $("#sbmt").bind("click", function () {
            var pwd1 = $("#pwd1").val();
            var pwd2 = $("#pwd2").val();
            if (pwd1 != pwd2) {
                alert("Passwords do not match");
            }
            else if (pwd1 == '' || pwd2 == '') {
                alert("password field must have some value");
            }
            else if (pwd1 == pwd2) {
                alert("Both Passwords match");
            }
        })

In this code the jQuery function first collects the values of password1 and password2 on sunmission of the form. Then it matches both the values. If they match, alert message of matching passwords will be shown. If either of the field is empty then alert of empty field will be shown. if both the passwords match, then success message of matching passwords will be shown. 

    $('.link').on('mouseenter', function () {
        if ($('.tooltip').is(':visible')) {
            $('.tooltip').hide();
        }
        $(this).next().show();
    })

    $('.link').on('mouseout', function () {
        $(this).next().hide();
    })

In this code the snippet for tooltip is shown. When the mouse enters the 'link' class and if tooltip is visible then it is hidden, otherwise tooltip is shown. When the mouse leaves the field, the tooltip is again hidden. Here tooltip is next element to the 'link'. So here I have used this.next() to refer it.

And At the last, the password strength is checked by constantly updating the length of input, if it is above the standard bar, it has some strength. If it also contains some numbers, special chars, then strength is increased accordingly. At the end, the label of 'weak','good','strong' is given from the strength.
