Code Snippet

$(function () {

$("#srch").bind("click", function() {

    // pull in the new value

    var searchTerm = $("#searchtext").val();

    // remove/update any old highlighted terms

    $("body").removeHighlight();

    // disable highlighting if empty

    if (searchTerm) {

        // highlight the new term

        $("body").highlight(searchTerm);

    }

    });

});

Here jQuery first stores the value inserted in textbox in a variable. Thenk it removes all highlighted words from paragraphs through removeHighlight() function. After that it highlights the instances of inserted text through highlight() function.
