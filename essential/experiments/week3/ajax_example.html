Playing Around with AJAX-JQuery!!

About The Experiment

In this experiment I have tried to use AJAX through JQuery. Here I have made a JSONP request to retrieve some information from remote page.Here the .ajax() function calls that page and stores the values in variables to display them on current HTML page.

Code Snippet 

    $.ajax({
    url: 'http://api.joind.in/v2.1/talks/10889',
    data: {
    format: 'json'
    },
    error: function() {
    $('#info').html('<p>An error has occurred</p>');
    },
    dataType: 'jsonp',
    success: function(data) {
    var $title = $('<h1>').text(data.talks[0].talk_title);
    var $description = $('<p>').text(data.talks[0].talk_description);
    $('#info')
    .append($title)
    .append($description);
    },
    type: 'GET'
    });

Description

In the snippet above, I’m using the second form of $.ajax(), which allows to specify the URL to which the request is sent as a property (url) of the object literal. Joind.in’s API accepts a JSONP request, so in the code above I have used the data property to define the format’s type that I want to obtain from the server as required by the API. However, the latter requires this data as part of the query string of a GET request, hence I’m also specifying the type property setting GET as its value. Finally, I wrote an error callback to display a message in case of error, and a success callback to display the title and the description of the talk in case of success.
