Popup Experiment

About The Experiment

In this experiment I have tried creating a popup through jQuery, which blurs the background while it is open. You can test it by clicking on the button below.

Code Snippet

    $(document).ready(function () {
    
        // Initialize the plugin
        $('#my_popup').popup({
        transition: 'all 0.3s',
        scrolllock: true // optional
        });
     });

Description

Here I have used the plugin for popup. It contains of popup function which is called when the button is clicked. This popup function will check the inout param provided to it. If there is some object is provided, then it will change its default values and set it according to diven parameters: e.g. $('#popup').popup({'color':'blue'}). If some string is provided then it will call the particular method of popup. e.g.:$('#popup').popup('hide'). If no parameters are passed then it will show default values set by popup() function.
